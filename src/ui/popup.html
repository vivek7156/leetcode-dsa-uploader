Quick Setup Guide
  1. 1
    Create a GitHub Token
    Go to github.com → Settings → Developer Settings → Personal Access Tokens (Fine-grained) and select repo with Contents (Read & Write) permission.
  2. 2
    Enter your details
    Click the ⚙ gear icon at the top right to open Settings. Paste your token, GitHub username, and the target repo name.
  3. 3
    Submit on LeetCode
    Solve and submit a problem. The extension automatically pushes your solution with problem description to GitHub.
  4. 4
    Sync older solutions
    Use the Sync Historical section below to bulk-import your past accepted submissions.
